# Iron House Gym - Project Structure

## Overview
Professional fitness platform with 9 pages, modern UI, and comprehensive features.

## Technology Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Node.js + Express + MongoDB
- **Authentication:** JWT
- **UI Components:** Lucide React + Framer Motion

## Pages & Features
1. **Home** - Landing page with hero section
2. **Programs** - Training programs management
3. **Sessions** - Personal training booking
4. **Class Schedule** - Group fitness classes
5. **Nutrition** - Meal and macro tracking
6. **Workout Planner** - Custom workout routines
7. **Community** - Social features and challenges
8. **Achievements** - Gamification and badges
9. **Dashboard** - Personal stats and progress

## Key Files
- `frontend/src/App.jsx` - Main app with routing
- `frontend/src/components/Layout.jsx` - Navigation and layout
- `frontend/src/components/[Page].jsx` - Individual page components
- `backend/server.js` - API server
- `package.json` - Dependencies

## Setup Instructions
```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend  
cd backend
npm install
npm start
```

## Access URLs
- Frontend: http://localhost:5173
- Backend API: http://localhost:5000/api
